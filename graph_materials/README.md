# Graph Materials

## Structure
- `csv/`: exported CSV files used as graph inputs
- `scripts/eda/`: R scripts for EDA visuals
- `scripts/analytics/`: R scripts for analytics visuals
- `scripts/reports/`: R scripts for report visuals
- `scripts/validation/`: R scripts for validation visuals

## Notes
- Keep SQL export files in `csv/`
- Update the `csv_file` path inside the relevant R script before running it
- Use the section-specific script that matches the phase of the project
