# olist-sql-business-intelligence
End-to-end SQL analysis of 100k+ real Brazilian e-commerce orders across 6 business themes: business performance, customer behaviour, seller quality, product intelligence, logistics, and payments. Built with Microsoft SQL Server.

## Project Status
🔄 In Progress

## Dataset
Brazilian e-commerce data from Olist — 100k orders from 2016 to 2018  
Source: [Kaggle](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)

## Tools
- Microsoft SQL Server
- Git / GitHub
